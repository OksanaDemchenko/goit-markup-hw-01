# WebStudio – HTML Markup

This project is the first homework from the GoIT HTML/CSS course.  
It contains the markup of the WebStudio website using semantic HTML.

## 🧰 Technologies

- HTML5
- CSS3
- Git
- GitHub

## 🚀 Features

- Semantic HTML structure
- Responsive layout
- Sections: Hero, Features, Team, Portfolio, Footer
- Navigation menu

## 📦 How to use

Simply open the `index.html` file in your browser to view the project.
